# Piedra, papel o tijera - Python
Imprementación del clásico juego "piedra, papel o tijera" desarrollada en python.
Este proyecto consiste en una versión jugable en consola donde el usuario compite contra el computador.

## CARACTERÍSTICAS:
    * juego contra el computador.
    * sistema de rondas (mejor de N partidas).
    * Sistema de puntaje en tiempo real.

## CÓMO JUGAR:
    1. Define cuántas rondas quieres jugar.
    2. Ingresa tu jugada: piedra, papel o tijera.
    3. El juego continúa hasta que uno de los jugadores alcanza el número de victorias definido.
